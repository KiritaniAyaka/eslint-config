# @kiritaniayaka/eslint-config

- Single quotes
- No semi
- Tab as indent

# Usage

```shell
npm install @kiritaniayaka/eslint-config -D
# or
yarn add @kiritaniayaka/eslint-config -D
# or
pnpm add @kiritaniayaka/eslint-config -D
```

Then add it to your `.eslintrc`:

```json
{
	"extends": "@kiritaniayaka"
}
```
